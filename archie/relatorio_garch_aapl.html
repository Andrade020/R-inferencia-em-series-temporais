# %% [markdown]
# #  Relatrio de Volatilidade Condicional (GARCH) - Apple (AAPL)
# 
# **Perodo:** 2018-01-01 a 2024-01-01
# 
# ---
# 
# ## 1. Configurao e Bibliotecas
# 
# Este relatrio utiliza o modelo **GARCH(1, 1)** com a distribuio **Standardized Skew Student's t** (t de Student Assimtrica) para estimar o risco (volatilidade) dos retornos dirios da Apple (AAPL).

# %%
import pandas as pd
import numpy as np
import yfinance as yf
from arch import arch_model
from statsmodels.stats.diagnostic import acorr_ljungbox
from statsmodels.stats.stattools import jarque_bera
import matplotlib.pyplot as plt
import matplotlib.dates as mdates

# Configuraes do modelo
TICKER = "AAPL"
START_DATE = "2018-01-01"
END_DATE = "2024-01-01"
DISTRIBUTION = "skewt"
MODEL_ORDER = (1, 1)

# Download e clculo dos retornos
data = yf.download(TICKER, start=START_DATE, end=END_DATE, auto_adjust=False)
returns = 100 * np.log(data['Adj Close'] / data['Adj Close'].shift(1)).dropna()

# %% [markdown]
# ## 2. Anlise Descritiva e Justificativa do Modelo
# 
# A **Curtose** (Kurtosis) e a **Assimetria** (Skewness) dos retornos justificam a escolha de uma distribuio no-Normal para o modelo GARCH, confirmando a alta probabilidade de eventos extremos.

# %%
# Estatsticas Descritivas
skewness = returns.skew().item()
kurtosis_excess = returns.kurtosis().item()
kurtosis = kurtosis_excess + 3

print("### Estatsticas Chave dos Retornos Dirios")

# CORREO FINAL: Cria o DataFrame manualmente a partir de um dicionrio, evitando o erro to_frame().T
desc_series = returns.describe()
stats_dict = desc_series.to_dict() # Converte a Series de estatsticas para dicionrio
stats_dict['Skewness'] = skewness
stats_dict['Kurtosis'] = kurtosis

# Cria o DataFrame de uma linha a partir do dicionrio
desc_df = pd.DataFrame(stats_dict, index=['AAPL']) 

print(desc_df.to_markdown(floatfmt=".4f"))

# Plotagem dos Retornos
plt.figure(figsize=(14, 4))
returns.plot(title=f'Retornos Logartmicos Dirios ({TICKER})')
plt.ylabel('Retorno (%)')
plt.grid(True)
plt.show()

# %% [markdown]
# ## 3. Estimao do Modelo GARCH(1, 1)
# 
# A alta persistncia ($\alpha_1 + \beta_1 \approx 0.9754$) e o baixo grau de liberdade ($\eta \approx 5.8$) so as principais descobertas.

# %%
# Estimao do Modelo GARCH
model = arch_model(
    returns,
    mean='Constant',
    vol='Garch',
    p=MODEL_ORDER[0],
    q=MODEL_ORDER[1],
    dist=DISTRIBUTION
)
results = model.fit(disp='off')
print(results.summary())

# %% [markdown]
# ## 4. Visualizao da Volatilidade Condicional
# 
# O grfico ilustra o **Agrupamento de Volatilidade** (*Volatility Clustering*), onde a volatilidade estimada ($\hat{\sigma}_t$) aumenta drasticamente em perodos de estresse (e.g., incios da pandemia, crises de inflao). 

# %%
# Plotagem da Volatilidade Condicional
conditional_volatility = np.sqrt(results.conditional_volatility)

fig, ax = plt.subplots(figsize=(14, 6))
ax.plot(returns.index, returns, label='Retornos', alpha=0.5, color='gray')
ax.plot(conditional_volatility.index, conditional_volatility, color='red', label='Volatilidade Condicional ($\hat{\sigma}_t$)')

ax.xaxis.set_major_formatter(mdates.DateFormatter('%Y-%m-%d'))
ax.set_title(f'Volatilidade Condicional (GARCH{MODEL_ORDER}) Skew-t vs. Retornos - {TICKER}')
ax.set_ylabel('Percentual (%)')
ax.legend()
ax.grid(True)
plt.show()

# %% [markdown]
# ## 5. Testes de Diagnstico
# 
# Os **altos p-valores** no Ljung-Box para os resduos ao quadrado confirmam que o modelo capturou toda a dependncia condicional.

# %%
std_resid = results.std_resid

# Ljung-Box para Mdia (z_t) e Varincia (z_t^2)
ljung_box_resid = acorr_ljungbox(std_resid, lags=[10, 20], return_df=True)
ljung_box_sq_resid = acorr_ljungbox(std_resid**2, lags=[10, 20], return_df=True)

print("### Teste de Ljung-Box para Resduos ao Quadrado (z_t^2)")
print("H0: No h autocorrelao remanescente na varincia (Volatilidade Modelada)")
print(ljung_box_sq_resid.to_markdown(floatfmt=".4f"))

# Teste de Jarque-Bera (Normalidade)
jb_test = jarque_bera(std_resid)
print("\n### Teste de Jarque-Bera para Normalidade")
print(f"Estatstica JB: {jb_test[0]:.4f}, p-valor: {jb_test[1]:.4f}")
print("\n**Concluso dos Testes:** O modelo  estatisticamente adequado, pois a hiptese de no-autocorrelao na volatilidade  aceita (p-valores > 0.05).")
